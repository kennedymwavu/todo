# todobend

<!-- badges: start -->
<!-- badges: end -->

A todo list backend. Built using [plumber](https://www.rplumber.io/).

## Installation

You can install the development version of todobend from
[GitHub](https://github.com/) with:

``` r
remotes::install_github("kennedymwavu/todobend")
```

## Usage

``` r
todobend::run_api()
```
